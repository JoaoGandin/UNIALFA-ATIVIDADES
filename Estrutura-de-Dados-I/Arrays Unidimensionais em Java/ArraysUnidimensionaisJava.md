# Arrays Unidimensionais em Java
Os Arrays unidimensionais são os que possuem apenas um índice para acessar seu conteúdo eles são declarados da seguinte forma:
```
tipo de dado[] nome do array = new tipo-de-dado[quantidade];
```

**tipo de dado** -> pode ser qualquer tipo de variável.
**nome do array** -> um nome válido; As mesmas regras para nomes de variáveis.
**quantidade** -> quantidade de elementos que o array pode manipular.

Os arrays podem ser criado e inicializados de outra maneira, invés de utilizar o operador new, é possível definir os elementos do array entre **chaves** e separados por vírgula. Esses elementos devem ser do mesmo tipo da variável definida como array. A sintaxe é:
```
Tipo-de-dao[] nome-do-array = { valores separados por vírgula }
```

---

Exemplos

```java
int[] numeros = new int[10];
```
Cria um array com o nome números que contém 10 elementos do tipo int e seu índice varia de 0 a 9.

```java
double[] precos = new double[5];
```
Cria um array com o nome precos que contém 5 elementos do tipo double e seu índice varia de 0 a 4.

```java
char[] sexo = new char[2];
```
Cria um array com o nome sexo que contém 2 elementos do tipo char e seu índice varia entre 0 e 1.

```java
boolean[] opcoes = new boolean[2];
```
Cria um array com o nome que contém 2 elementos do tipo boolean e seu índice varia entre 0 e 1.

```java
String[] meses = new String[12];
```
Cria um array com o nome meses que contém 12 elementos do tipo String e seu índice varia de 0 a 11.
